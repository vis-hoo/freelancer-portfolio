---

## Project naming convention

Folders follow the pattern `project-XX-short-description`, where `XX` is a zero-padded number (01, 02, 03...). This keeps projects sorted chronologically and easy to navigate.

Examples:
- `project-01-ecommerce-orders-cleanup`
- `project-02-employee-csv-automation`
- `project-03-monthly-sales-sql-report`

---

## Notes template

Each project contains a `notes.md` with this structure:

```markdown
## Task
Brief description of what needed to be done.

## Tools
- e.g. Microsoft Excel, Python 3.x, SQLite

## What was done
Step-by-step or summary of the work performed.

## Result
What the deliverable looked like and whether the client was satisfied
(for practice projects: what the exercise demonstrated).
```
